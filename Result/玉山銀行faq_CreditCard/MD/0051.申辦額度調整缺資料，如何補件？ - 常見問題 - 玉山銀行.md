# 申辦額度調整缺資料，如何補件？

申請調高永久額度，可透過以下方式補交財力證明：

  * 官網：信用卡影像上傳系統。
  * 信用卡額度調整財力證明補件QR CODE  

  * 行動銀行：申請資料上傳，即可補件。

上傳資料後，畫面會顯示上傳成功，再請等候審核結果，有其它問題會再以簡訊或電話通知。

  * 信用卡
  * 申請
  * 行動銀行
  * 操作設定
