# 收到逾期簡訊通知，是否會有利息產生？

信用卡帳單逾期繳款依約定計付循環信用利息及違約金。  
若您在繳款截止日後，「第一次」收到提醒繳款簡訊，請於收到簡訊當天完成繳款且隔日完成銷帳，即不會收取循環利息與違約金，提醒您記得準時繳款，避免影響個人信用。
